## 📝 价值等级系统

### 等级定义

| 等级 | 价值 | 颜色 | 字体 | 音量 | 地图图标 |
|------|-------|-------|------|--------|----------|
| **S** | 顶级价值 | 金色 (255,215,0) | 45 (最大) | 300 (最大) | Size 2 星形 |
| **A** | 高价值 | 橙色 (255,165,0) | 40 (大) | 250 (清晰) | Size 1-2 菱形 |
| **B** | 中高价值 | 蓝色 (0,191,255) | 35 (标准) | 200 (弱提示) | Size 1 圆形 |
| **C** | 中等价值 | 绿色 (50,205,50) | 30 (小) | 150 (极弱) | 无 |
| **D** | 低价值 | 灰色 (105,105,105) | 25 (最小) | 无 | 无 |
| **E** | 无价值 | 隐藏 | - | - | - |

### 音效设计原则

**S级物品**（顶级价值）
- 音量: 300（最大）
- 特点: 强烈、短促、极易识别
- 示例: 混沌石、崇高石、神圣石、瓦尔宝珠、完美通货

**A级物品**（高价值）
- 音量: 250
- 特点: 清晰、有存在感
- 示例: 富豪石、增幅石、传奇装备（60+）、高品级稀有装备

**B级物品**（中高价值）
- 音量: 200
- 特点: 弱提示音
- 示例: 蜕变石、中等级引路石、5连装备

**C级物品**（中等价值）
- 音量: 150
- 特点: 极弱或可选
- 示例: 碎片、低等级引路石、4连装备

**D级物品**（低价值）
- 音量: 不播放
- 示例: 低品级装备、3连装备

**E级物品**（无价值）
- 音量: 不播放
- 处理: 隐藏
