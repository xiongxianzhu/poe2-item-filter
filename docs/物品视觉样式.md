## 📝 使用说明

### 如何自定义样式

1. **修改颜色**：
   - 调整 RGB 值（0-255）
   - 调整 Alpha 透明度（0-255）

2. **修改字体**：
   - 范围：1-45
   - 建议使用上述推荐值

3. **修改地图图标**：
   - Size: 0-2
   - Shape: Circle, Diamond, Star, Triangle, Cross 等
   - Color: Red, Green, Blue, Yellow, Orange, Pink, Purple, Cyan 等

4. **修改音量**：
   - 范围：0-300
   - 建议：S(300) > A(250) > B(200) > C(150)

### 注意事项

- 所有颜色值必须在 0-255 之间
- Alpha 透明度：0=完全透明，255=完全不透明
- 地图图标 Size 2 比 Size 1 大约 50%
- 声音文件必须是 MP3 格式
- 相对路径引用：`sounds/category/filename.mp3`
