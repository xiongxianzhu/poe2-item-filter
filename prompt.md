====================
一、总体目标
====================

1. 过滤器必须基于【物品实际价值】进行分级，而不是仅依据稀有度
2. 适用于【通用刷图 / 赛季初到中后期】
3. 核心目标：
   - 高价值物品：强提示、一眼可见
   - 中价值物品：明确但克制提示
   - 低价值物品：最大化降噪
4. 支持：
   - 自定义声音（CustomAlertSound）
   - 地面图标（MinimapIcon）
   - 光柱特效（PlayEffect）
5. 风格要求：
   - 专业
   - 清晰
   - 稳定
   - 可长期维护
6. 输出文件名：
   - 必须严格遵循：xx-0.4.filter
   - 文件编码：UTF-8

====================
二、物品价值等级（共 6 级，必须严格使用）
====================

过滤器中的所有物品，必须明确归属以下 6 个价值等级之一，
不得新增或合并等级：

【S 级 - 顶级价值】

- 赛季核心通货
- 极高价值传奇 / 独特物品
- 顶级制作或赛季专属材料
- 极低掉率但高需求物品

【A 级 - 高价值】

- 主流 Build 强需求物品
- 高价值稀有装备（优秀基底 / 高潜力词缀）
- 高流通通货

【B 级 - 中高价值】

- 可交易、需求稳定的装备或材料
- 中后期常用制作资源
- 市场价值明确但非顶级

【C 级 - 中等价值】

- 可过渡装备
- 市场价值一般，但仍具实用性
- 特定流派或阶段有需求的物品

【D 级 - 低价值】

- 练级期或极少数场景可用
- 市场价值较低，主要作为补充资源
- 提示应明显弱化

【E 级 - 无价值 / 噪音】

- 明显无交易价值或实用价值的掉落
- 不适合当前赛季环境
- 默认应隐藏

====================
六、POE2 官方过滤器语法规范
====================

POE2 官方过滤器语法: <https://www.pathofexile.com/item-filter/about>

### 1. 基本语法结构

Show / Hide / Minimal 块结构：

```
Show
    <条件1>
    <条件2>
    <动作1>
    <动作2>

Hide
    <条件1>
    <动作1>
```

### 2. 常用条件（POE2 相关）

**基础条件：**
- BaseType "<物品名称>" - 基础类型名称
- Class "<类别名称>" - 物品类别
- Rarity <Normal/Magic/Rare/Unique> - 稀有度
- ItemLevel <数字> - 物品等级
- DropLevel <数字> - 掉落等级
- AreaLevel <数字> - 区域等级

**插槽相关：**
- Sockets <数字> <颜色> - 插槽数量（R/G/B/A/D/W）
- LinkedSockets <数字> - 连接插槽数量
- SocketGroup <数字> <颜色组合> - 插槽组合

**POE2 特有条件：**
- UnidentifiedItemTier <数字> - 未鉴定物品等级层级
- WaystoneTier <数字> - 引路石层级
- TransfiguredGem "<宝石名称>" - 变化宝石
- Corrupted <True/False> - 已腐化
- Identified <True/False> - 已鉴定

**其他条件：**
- Width <数字> / Height <数字> - 物品尺寸
- StackSize <数字> - 堆叠数量
- HasEnchantment "<附魔名称>" - 附魔
- HasInfluence <Shaper/Elder/等> - 影响之力

### 3. 常用动作

**声音动作：**
```
CustomAlertSound "sounds/混沌石.mp3" 300
```

**视觉动作：**
- MinimapIcon <大小> <颜色> <形状>
  - 大小：0-2（2最大）
  - 颜色：Red, Green, Blue, Brown, White, Yellow, Cyan, Grey, Orange, Pink, Purple
  - 形状：Circle, Diamond, Hexagon, Square, Star, Triangle, Cross, Moon, Raindrop, Kite, Pentagon, UpsideDownHouse

- PlayEffect <颜色> <Temp>
  - 颜色：Red, Green, Blue, Brown, White, Yellow, Cyan, Grey, Orange, Pink, Purple
  - Temp：临时显示（省略为永久）

- SetBorderColor <R> <G> <B> <Alpha>
- SetTextColor <R> <G> <B> <Alpha>
- SetBackgroundColor <R> <G> <B> <Alpha>
- SetFontSize <大小>  # 1-45

### 4. 示例规则

```
# S级 - 混沌石
Show
    Class "Currency"
    BaseType "混沌石"
    SetBorderColor 255 215 0 255
    SetBackgroundColor 255 215 0 50
    SetFontSize 45
    MinimapIcon 2 Yellow Circle
    PlayEffect Yellow Temp
    CustomAlertSound "sounds/混沌石.mp3" 300
```

### 5. 注意事项

- 所有条件必须在同一 Show/Hide 块中
- 多个条件之间为 AND 关系（必须全部满足）
- 颜色值为 RGB 格式，范围 0-255
- Alpha 透明度：0-255（0完全透明，255完全不透明）
- 注释使用 # 开头
- 每个规则块之间建议用空行分隔

====================
七、地面图标（Minimap Icon）规则
====================

- S 级：必须显示，远距离可识别（Size 2）
- A 级：必须显示，中远距离可识别（Size 1-2）
- B 级：可选显示（Size 1）
- C 级：默认不显示
- D / E 级：不显示

图标需体现价值差异（颜色或形状）。
