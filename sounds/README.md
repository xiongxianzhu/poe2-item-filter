## 声音文件要求

### 文件格式
- **格式**: MP3
- **命名**: <物品简体中文名>.mp3
- **编码**: UTF-8（文件名）

### 声音设计原则

#### S级物品（顶级价值）
- 音量: 300（最大）
- 特点: 强烈、短促、极易识别
- 示例: 混沌石、崇高石、神圣石、瓦尔宝珠

#### A级物品（高价值）
- 音量: 250
- 特点: 清晰、有存在感
- 示例: 改造石、增幅石、传奇装备

#### B级物品（中高价值）
- 音量: 200
- 特点: 弱提示音
- 示例: 魔法石、中等级界石、5连装备

#### C级物品（中等价值）
- 音量: 150
- 特点: 极弱或可选
- 示例: 碎片、低等级界石、4连装备

#### D级物品（低价值）
- 音量: 不播放
- 示例: 低品级装备

#### E级物品（无价值）
- 音量: 不播放
- 处理: 隐藏

## 使用方法

### 1. 创建目录结构

在 POE2 过滤器所在目录创建 sounds 文件夹及其子目录：

```bash
# Windows
mkdir sounds
mkdir sounds\currency
mkdir sounds\unique
mkdir sounds\crafting
mkdir sounds\fragment
mkdir sounds\other
```

### 2. 准备声音文件

将声音文件放入对应目录，确保文件名与过滤规则中引用的名称完全一致。

**示例**:
- `sounds/currency/混沌石.mp3`
- `sounds/unique/传奇项链.mp3`

### 3. 验证

启动 POE2，加载过滤器 `poe2-0.4.filter`，测试声音是否正常播放。

## 音效资源建议

### 推荐来源
- POE 官方音效提取
- 自行录制/制作
- 第三方音效库（需确认版权）

### 音效特点
- 时长: 建议 1-2 秒
- 音质: 清晰、无噪音
- 风格: 统一、专业

## 注意事项

1. **文件名大小写**: Windows 不区分，但建议保持一致
2. **路径引用**: 过滤器中使用相对路径 `sounds/...`
3. **缺失文件**: 如果声音文件不存在，游戏不会报错，但不会播放声音
4. **音量平衡**: 确保不同物品的声音音量协调，避免某些过响
5. **性能影响**: 自定义声音数量不宜过多，避免影响游戏性能

## 故障排查

### 声音不播放
- 检查文件路径是否正确
- 确认文件名拼写（包括扩展名）
- 验证 MP3 文件是否损坏
- 检查游戏音效设置

### 声音音量异常
- 调整过滤规则中的音量参数（0-300）
- 检查 MP3 文件本身音量
- 调整游戏主音量
